## Roadmap

### Version 1.1 (Q1 2026)
- **Export-Funktionen**
  - Export als TXT, DOCX, PDF
  - Batch-Export mehrerer Aufnahmen
  - Clipboard-Integration für schnelles Kopieren
- **UI-Verbesserungen**
  - Dunkles/Helles Theme-Umschaltung
  - Anpassbare Schriftgrößen
  - Drag & Drop für Audio-Dateien
- **Aufnahme-Features**
  - Pause/Resume während der Aufnahme
  - Audio-Qualitätseinstellungen (Bitrate, Sample-Rate)
  - Rauschunterdrückung

### Version 1.2 (Q2 2026)
- **Mehrsprachigkeit**
  - Automatische Spracherkennung
  - Unterstützung für 20+ Sprachen
  - Echtzeit-Übersetzung
- **Erweiterte KI-Features**
  - Sentiment-Analyse
  - Keyword-Extraktion
  - Automatische Kategorisierung
  - Sprechererkennung (Diarization)
- **Cloud-Integration**
  - Optional: Cloud-Backup für Aufnahmen
  - Synchronisation zwischen Geräten
  - Team-Sharing-Funktionen

### Version 1.3 (Q3 2026)
- **Produktivitäts-Features**
  - Tags und Kategorien für Aufnahmen
  - Volltextsuche über alle Transkriptionen
  - Favoriten und Bookmarks
  - Notizen zu Aufnahmen hinzufügen
- **Erweiterte Prompt-Templates**
  - Benutzerdefinierte Prompt-Bibliothek
  - Template-Sharing in der Community
  - Prompt-Variablen und Platzhalter
- **Integrationen**
  - Notion, Obsidian, Evernote Export
  - Slack/Discord Bot-Integration
  - API für Drittanbieter

### Version 2.0 (Q4 2026)
- **Plattform-Erweiterung**
  - macOS native App
  - Linux native App
  - Mobile Apps (iOS/Android)
- **Echtzeit-Transkription**
  - Live-Transkription während der Aufnahme
  - Streaming-Modus für lange Meetings
  - WebSocket-basierte Echtzeit-Updates
- **Kollaboration**
  - Gemeinsame Workspaces
  - Kommentare und Annotations
  - Versionskontrolle für Transkriptionen
- **Enterprise-Features**
  - Self-Hosted Option
  - SSO/SAML-Integration
  - Audit-Logs und Compliance
  - Custom LLM-Endpoints

### Langfristige Vision
- **KI-Assistent-Modus**: Proaktive Vorschläge während der Aufnahme
- **Meeting-Modus**: Automatische Agenda-Erkennung und Action-Items
- **Voice-Commands**: Steuerung der App per Sprachbefehl
- **Offline-First**: Vollständige Funktionalität ohne Internetverbindung
- **Plugin-System**: Erweiterbarkeit durch Community-Plugins
